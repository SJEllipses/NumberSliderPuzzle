# Number Slider Puzzle
A simple 3x3 Number Slider Puzzle game based on OpenCV.

  
## Requirements
1. CMake
2. OpenCV Release

  
## How to build
```bash
cd build
cmake -G "MinGW Makefiles"
make
```
It also supports MSVC.
